html메소드와 text메소드
html메소드 : 태그를 포함
text메소드 : 태그를 포함하지 않는다

alert($("body").html()); // body 태그 안의 모든 태그를 포함한 문장
alert($("body").text()); // body 태그안의 모든 문자만
alert($("p").text()); //여러개인경우 모든 p태그의 문자만
alert($("p").html()); //p가 여러개인경우 첫번째 인거만 태그를 포함한 문장
