type : 입력하는 데이터의 형태의 속성(텍스트? 파일? 이미지?) 
		text : 한줄입력(줄바꿈 불가)
		radio : 여러개 중 한개를 선택
		checkbox : 여러개 선택 or 한개 선택 or 선택이 없을 수 있다 
		hidden : 화면에서는 보이지 않는다. 값은 서버로 전송된다
		image : 전송 버튼 역할을 한다. submit과 같은 역할
		button : 이벤트를 부여해서 전송 역할을 한다
		 
	name : 서버로 전송 시 입력한 데이터로 보내기 위한 속성
	(name이 있어야 jsp에서 제대로 받을 수 있다)
	textarea 태그(또는 요소)
	button 태그(요소)
		type을 지정하지 않으면submit(전송)역할을 수행한다
