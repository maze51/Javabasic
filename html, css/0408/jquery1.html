jquery : javascript의 라이브러리 입니다.
javascript를 좀 더 간편하게 사용할 수 있습니다.
javascript를 대체하는 것은 아닙니다.

다운르도 방식 : jQuery.com에서 다운로드
<script src="../js/jquery-3.3.1.min.js"></script> CDN방식 : 직접 다운받지 않고 실행시마다 네트워크를 통해서 공개서버로부터 자동으로 다운받아 html파일에 include한다
P태그를 선택하는 방법
javascript: document.getElementsByTagName('p')
jQuery: $('p')
