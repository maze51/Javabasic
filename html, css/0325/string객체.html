1. charAt(index) : 지정한 index위치의 글자를 얻는다



2. charCodeAt(index) : 지정한 index위치의 글자의 unicode값을 얻는다



3. indexOf("검색값", start) : 지정한 검색값의 위치를 얻는다. 검색은 start위치에서 시작한다



4. trim() : 문자열 앞 뒤 공백을 제거한다



5. concat() : 문자열과 문자열을 합친다




6. substring(start, end) : start~end-1까지의 문자열을 추출한다

이미지 위에 마우스를 올려놓으면 title이 뜬다
lastIndexOf('/'), lastIndexOf('.'),
getAttribute('src'), setAttribute('title', '값')
substring(start, end)



7. slice(start, end) : start~end-1까지의 문자열을 자르기 추출한다




8. substr(start, length) : start에서부터 지정된 length만큼의 문자를 추출한다




9. split(구분자) : 구분자에 의해 문자열을 분리해서 배열로 만든다
